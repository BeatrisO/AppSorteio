# AppSorteio

É um aplicativo Android simples e rápido desenvolvido em Kotlin, criado para gerar números aleatórios com apenas um toque.
Ideal para sorteios rápidos, jogos, decisões ou qualquer situação em que seja necessário obter um número de forma totalmente automática.

### Funcionalidades

* Geração instantânea de números aleatórios
* Intervalo configurável (valor inicial e valor final)
* Interface limpa e intuitiva
* Botão dedicado para realizar o sorteio
* Exibição clara do resultado na tela
* Animação 

### Tecnologias Utilizadas

* Kotlin
* Android Studio
* Jetpack Compose (projeto começou usando ViewBinding)

### Melhorias Futuras
* Histórico de números sorteados
* Tema claro/escuro
